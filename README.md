# AI News Aggregator

An intelligent news aggregation service that fetches articles from multiple sources (Reddit, HackerNews, NewsData), analyzes them using Groq AI to score and categorize them, and serves the top trending tech news.

## Features

- **Multi-source Aggregation**: Fetches news from Reddit, HackerNews, and NewsData.
- **AI Analysis**: Uses Groq AI to analyze, summarize, and score articles based on relevance and quality.
- **Smart Search**: Search through cached and live news sources with AI scoring.
- **Category Filtering**: Filter news by topics like AI/ML, Web3, FinTech, etc.
- **Trending Algorithm**: Filters and ranks news to show only the most important updates.
- **Caching**: Uses Redis to cache raw and filtered results for performance.
- **Rate Limiting & Security**: Implements Helmet and Express Rate Limit.

## Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **AI**: Groq SDK
- **Database/Cache**: Redis
- **Logging**: Winston
- **Testing**: Jest

## Prerequisites

- Node.js (v18+)
- Redis Server
- Groq API Key
- NewsData API Key (optional)
- Reddit/HackerNews (public APIs)

## Setup

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd ai-news-aggregator
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Environment Variables**
    Create a `.env` file in the root directory:
    ```env
    PORT=3000
    REDIS_URL=redis://localhost:6379
    GROQ_API_KEY=your_groq_api_key
    NEWSDATA_API_KEY=your_newsdata_key
    LOG_LEVEL=info
    ```

4.  **Run the application**
    ```bash
    # Development mode
    npm run dev

    # Production start
    npm start
    ```

5.  **Run Frontend**
    Open a new terminal:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
    The app will be available at `http://localhost:5173`.

6.  **Run Tests**
    ```bash
    npm test
    ```

## API Endpoints

### Get Trending News
`GET /api/v1/trending`

Returns a list of AI-curated trending news articles.

**Response:**
```json
[
  {
    "title": "Example News Title",
    "url": "https://example.com/news",
    "source": "HackerNews",
    "ai_summary": "A brief summary generated by AI...",
    "ai_score": 8.5,
    "ai_category": "AI/ML"
  },
  ...
]
```

## Project Structure

- `src/services/fetchers`: Modules to fetch data from external APIs.
- `src/services/aggregator.js`: Combines data from all fetchers.
- `src/controllers/trendingController.js`: Handles the AI filtering and response logic.
- `tests/`: Jest test suites.
